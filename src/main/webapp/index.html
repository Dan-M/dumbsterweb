Welcome to Dumbster Web

Dumbster Web is a simple web application that wraps Dumbster, a fake SMTP Server. It's purpose is to support integration testing of applications that need to send emails.

If you see this page, a SimpleSmtpServer has already been started. Configure the application to be tested to use
  • mail.smtp.host = hostName
  • mail.smtp.port = port
and deploy it for system / integration testing.

You can now either Test the SMTP server by sending a mail to it from this application or you can check the Mails that the server has already recieved.
